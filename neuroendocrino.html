Sistema Nervioso

S.N. Central (SNC)

  • • Encéfalo: Centro de procesamiento y toma de decisiones.
  • • Médula: Vía de conducción de impulsos.
GIRAR PARA INTERRELACIÓN

Relación Nervio-Hormona

El SNC detecta estímulos del entorno (ej: luz, peligro). La amígdala envía señales al hipotálamo para iniciar la respuesta química. El cerebro además "lee" la sangre para ajustar su conducta.

S.N. Periférico (SNP)

Red de nervios que conecta el SNC con los receptores sensoriales y los órganos efectores del cuerpo.

GIRAR PARA INTERRELACIÓN

Relación Nervio-Hormona

El Sistema Autónomo (SNP) estimula de forma directa glándulas como la Médula Suprarrenal, permitiendo una liberación química casi instantánea ante el estrés.

GIRAR PARA DETALLE

El Punto de Unión

Es el verdadero "cerebro endocrino". Recibe información neuronal del SNC y responde liberando neurohormonas que controlan a la hipófisis. Monitoriza la sangre constantemente.

Hipófisis

Libera hormonas tróficas (TSH, ACTH, LH, FSH) directamente al torrente sanguíneo.

GIRAR PARA DETALLE

Amplificador

Funciona como una antena emisora. Recibe señales locales del hipotálamo y las convierte en una señal sistémica masiva que llega a todas las glándulas efectoras.

FEEDBACK NEGATIVO

Cuando el nivel hormonal en sangre es alto, el Hipotálamo detecta la señal (humoral) e inhibe la orden, manteniendo el equilibrio (homeostasis).

2

El Hipotálamo actúa como bisagra integrando ambas señales.

3

La Hipófisis amplifica y distribuye el mensaje por la sangre.

4

Las hormonas retornan e inhiben el sistema para evitar el exceso.
